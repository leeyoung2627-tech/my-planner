PDF — 버튼을 누르면 인쇄 창이 열려요. 대상에서 "PDF로 저장"을 선택하세요.
Word(.docx) — 워드에서 바로 열리고, 구글 문서에 업로드해도 열려요.
JPEG — 세로로 긴 문서를 A4 크기 단위로 자동 분할하여 하나의 ZIP 파일로 저장해 드립니다.
구글 드라이브 / Dropbox — 백업 파일(.json)이 자동으로 내려받아지고, 해당 서비스가 새 탭에서 열려요. 열린 창에 파일을 업로드하면 클라우드에 저장됩니다.
